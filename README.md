Scame
=====

An Emacs configuration.

## Dependencies

- [Emacs 24](http://wikemacs.org/wiki/Installing_Emacs)
- [Cask](https://github.com/cask/cask)


## Installation

### Manual

    $ mkdir $HOME/.emacs.d
	$ cd $HOME/.emacs.d
	$ git clone https://github.com/nlamirault/scame
    $ cask install
